Frontend development internship task 1 - image Gallery

## 📱 Project Name: Image Gallery

A premium, fully responsive Image Gallery built with modern web technologies during my Frontend Development Internship at CodeAlpha.

---

## ✨ Features
- **Grid Layout:** Beautiful responsive layout using CSS Grid and Flexbox.
- **Hover Effects:** Smooth zoom-in effects with sleek overlays on hover.
- **Lightbox View:** Click on any image to open it in full screen (Modal).
- **Navigation Controls:** Next and Previous buttons to easily cycle through images inside the lightbox.
- **Responsive Design:** Completely optimized for Mobile, Tablet, and Desktop screens.

---

## 🛠️ Tech Stack Used
- **HTML5:** For the structural layout of the gallery and lightbox modal.
- **CSS3:** For styling, dark-theme layout, media queries, and smooth animations.
- **JavaScript (ES6):** For the lightbox logic, event listeners, and navigation array.
